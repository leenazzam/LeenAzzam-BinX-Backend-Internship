# Day 3 

Today I modeled a small library domain with two related classes, Book and Member.

I added a record type, BorrowRequest, as an immutable data transfer object for borrowing details.

I applied encapsulation using private backing fields and public properties, with constructors that validate the input.

I defined an interface, INotifiable, implemented by two unrelated classes, and wrote a method that demonstrates polymorphism by accepting the interface type.

I committed the domain model to my GitHub repository with a clear commit message.
