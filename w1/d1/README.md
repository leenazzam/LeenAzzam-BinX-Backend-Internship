# Day 1: Environment Setup & First Console App

Today I set up my .NET development environment by installing the .NET SDK and verifying it with `dotnet --version`. 
I configured VS Code with the C# Dev Kit extension, then created my first console application using `dotnet new console -o HelloBinX` and ran it with `dotnet run`.
 I modified the program to print my name and today's date instead of the default "Hello, World!" message, then rebuilt and reran it to confirm the change.
Finally, I created a GitHub repository named `Leen-Azzam-BinX-Backend-Internship` and pushed my first commit.
