# Week 1 
This week covered setting up the .NET development environment and practicing core C# fundamentals across 5 days.

Day 1 focused on installing the .NET SDK and building the first console app.

Day 2 covered value types, reference types, control flow, and nullable reference types.

Day 3 covered object-oriented programming: classes, records, encapsulation, and interfaces.

Day 4 covered collections, LINQ queries, async/await, and input validation.

Day 5 focused on the Git feature-branch workflow and opening this pull request.
