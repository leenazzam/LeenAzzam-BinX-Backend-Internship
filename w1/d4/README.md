
# Day 4

Today I created a list of 8 books using my Day 3 domain model.

I wrote three LINQ queries: one filter to get borrowed books, one projection to get just the titles, and one aggregation to calculate the average price.

I wrote an async method that simulates a delay using Task.Delay, and awaited it from Main.

I validated user input using TryParse to safely check if the entered value is a valid number.
